Habit Tracker Server

This is the server-side code for the Habit Tracker application. It is built using Node.js, Express, and MongoDB, with Firebase Admin integration for authentication verification.

Features

User authentication and authorization using Firebase Admin SDK.
CRUD operations for user habits.
Integration with MongoDB Atlas for database storage.
CORS enabled for frontend-backend communication.
RESTful API endpoints.

---

Technologies Used

Node.js- JavaScript runtime.
Express.js - Web framework for API routing.
MongoDB - NoSQL database.
Firebase Admin SDK - For authentication and user verification.
dotenv - For environment variable management.
CORS - Cross-Origin Resource Sharing.
nodemon(optional) - For development.
